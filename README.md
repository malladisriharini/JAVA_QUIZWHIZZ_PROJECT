# QuizWhizz Java Project

## 📌 Description

QuizWhizz is a Java-based quiz application developed using Swing and MySQL. It allows users to take quizzes, view scores, and track performance.

## 🚀 Features

* User Login & Registration
* Quiz System
* Leaderboard
* Admin Panel (Add/Delete Questions)
* Result & Statistics

## 🛠 Technologies Used

* Java (Swing)
* JDBC
* MySQL

## 🗄 Database Setup

1. Open MySQL Workbench
2. Create a database named `quizwhizz`
3. Import the file `quizwhizz.sql`

## ▶️ How to Run

1. Add MySQL Connector JAR
2. Open project in IDE
3. Run `Main.java`

## 📁 Project Structure

* dao → Database operations
* model → Data classes
* ui → User interface
* resources → Images
